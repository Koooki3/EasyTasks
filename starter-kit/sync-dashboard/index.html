How It Writes

1. This page reads and writes the markdown files in the starter vault directly.

2. Toggling a checkbox writes back to the original task line.

3. Fixed tasks are sourced from DOC/fixed-tasks.json.

4. Use one editing entry point at a time to reduce write conflicts.

Status Notes

1. Open tasks use - [ ].

2. Completed tasks use - [x] with done: YYYY-MM-DD.

3. Cancelled tasks can use - [-] and optionally cancel: YYYY-MM-DD.

4. Cancelled tasks are excluded from open and completed queues.
